This word cloud shows the words from the song "Change Gonna Come".

It was generated by:

  • Finding the most frequent words using a hashtable
  • The code was written in Java
  • Using the Kumo Library.
  • With the help of the Maven Package manager

These Key value pair could be of any data type and we can even use generics to insert any type of data into it.
